# Nicholas Jackson — Personal Website

Personal academic website for Nicholas Jackson, 3rd Year PhD Student in Biomedical Informatics at Vanderbilt University.

Built with pure HTML, CSS, and JavaScript — no frameworks, no build tools, no server required.

## Pages

- **Home** — Bio, research overview, links
- **Research** — Research directions + filtered publications & presentations
- **CV** — Embedded PDF viewer + download

## Local Development

Open `index.html` in any browser. No server needed.
